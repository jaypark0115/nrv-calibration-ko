# Jay 개인 페이지

Jay가 작성한 기술 글과 정리 자료를 모아두는 개인 페이지입니다.
현재는 NRV DVS 캘리브레이션을 처음 보는 사람도 흐름을 이해할 수 있도록,
캘리브레이션 설명 글 3개를 먼저 공개해두었습니다.

NRV DVS 캘리브레이션 설명은 Jay가 NRV 이벤트 카메라에 맞는 캘리브레이션 방법을
직접 찾아보고, 실제 설명 흐름에 맞게 다시 정리한 글 묶음입니다.

아래의 NRV DVS 캘리브레이션 글 3개는 NRV의 지원을 받아 작성한 한국어 설명 자료입니다.
이 저장소의 첫 화면은 Jay의 개인 글 목록이며, 앞으로 추가될 개인 논문, 프로젝트 기록,
실험 노트 전체가 NRV 공식 문서라는 뜻은 아닙니다.

같은 내용의 공식 영문 문서는 NRV docs 홈페이지에서 확인할 수 있습니다.
