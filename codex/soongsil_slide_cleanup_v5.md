# 숭실대 SoC/FPGA Bus 이미지/캡션 정리 기록 v5

작성일: 2026-07-12

## 수정 목적

이전 버전은 강의 PDF에서 렌더링한 이미지를 그대로 넣은 느낌이 강했다.
특히 이미지 안에 숭실대학교 로고, 슬라이드 제목, 푸터가 함께 보이고, 캡션도 “무엇을 설명한 슬라이드”처럼 붙여 넣은 문장으로 보일 수 있었다.
이번 수정에서는 사용자의 피드백에 맞춰 사이트 본문이 직접 정리한 기술 노트처럼 읽히도록 이미지와 문장을 정리했다.

## 반영 내용

- PDF에서 가져온 이미지 17개를 다시 원본 PDF에서 렌더링한 뒤, 도식/타이밍도 중심으로 크롭했다.
- 숭실대학교 로고, 슬라이드 제목, 하단 푸터가 보이지 않도록 각 이미지별 crop box를 조정했다.
- topology 이미지에 남은 제목 잔여 조각은 흰 배경으로 덮어 제거했다.
- PDF 이미지 아래의 긴 figcaption을 삭제했다.
- `수업자료 - ...`, `...를 설명한 슬라이드`처럼 붙여넣기 느낌이 나는 figure label과 alt text를 짧은 기술 라벨로 바꿨다.
- `AMBA bus protocol summary` 이미지는 슬라이드 캡처 느낌이 강해 AHB 페이지 본문에서 제거했다.
- 잘못 자동 치환된 필기 이미지 라벨을 `AHB / AXI 사용 기준 필기`로 복구했다.

## 남긴 방향

이미지는 이제 문서의 주 설명이 아니라, 본문에서 설명한 RTL/버스 개념을 보조하는 도식으로만 보이도록 했다.
설명은 이미지 밑에 길게 붙이지 않고, 이미지 앞뒤 본문과 코드 블록에서 처리한다.
