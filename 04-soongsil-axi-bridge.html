SoC/FPGA Bus 학습 노트

이 노트는 한 페이지짜리 요약문이 아니라 APB, AHB, AXI, AXI-to-APB Bridge로 나누어 정리했습니다. 각 페이지는 수업자료에서 다룬 개념, Lab별 RTL/TB 실습, 파형 검증 이미지를 함께 보도록 구성했습니다.

01

APB Control Bus

Lab2~4의 register, SRAM, interrupt 실습을 설명합니다.

 02

AHB Pipelined Bus

Lab5~6의 AHB register/SRAM 실습과 HREADY timing을 설명합니다.

 03

AXI Channel Bus

Lab7의 AXI slave channel handshake를 설명합니다.

 04

AXI-to-APB Bridge

최종 protocol conversion RTL과 PASS 검증을 설명합니다.

수업자료 기준 Lab 진행 흐름

숭실대 강의자료의 흐름은 환경 구축에서 시작해 APB, AHB, AXI를 차례로 다루고, 마지막에는 AXI 요청을 APB register access로 변환하는 프로젝트로 이어졌습니다. 아래 흐름은 각 개념 페이지가 어떤 Lab 근거를 바탕으로 작성되었는지 보여 주기 위한 요약입니다.

Lab1

환경 및 Testbench

Linux, Xcelium, SimVision, Verilog testbench 실행 흐름을 확인했습니다.

Lab2~4

APB 기반 제어

Register access에서 SRAM, interrupt/mask 구조까지 APB 제어 범위를 확장했습니다.

Lab5~6

AHB 기반 접근

AHB register access와 SP-SRAM 접근을 통해 phase pipeline과 HREADY timing을 확인했습니다.

Lab7 / Project

AXI와 Bridge

AXI slave channel handshake를 검증한 뒤 AXI-to-APB Bridge 최종 프로젝트로 연결했습니다.
